# Nvim

> this is my personal neovim configuration

## Install

### prerequisite

- [nodejs](https://nodejs.org/)
- [cmake]()
- [fzf](https://github.com/junegunn/fzf)
- [ripgrep](https://github.com/BurntSushi/ripgrep)
- [neovim]()

### setup

#### linux

```bash
    cd ~/.config
    git clone https://github.com/shadinmhd/nvim
    nvim
```

#### windows

```bash
    cd $env:LOCALAPPDATA
    git clone https://github.com/shadinmhd/nvim
    nvim
```
